# ghostty

## Windows

```powershell
mkdir -p $env:USERPROFILE\ghostty\.config\; vim $env:USERPROFILE\ghostty\config

```

## UNIX
```shell
mkdir -p $HOME/.config/ghostty && vim $HOME/.config/ghostty/config
```
